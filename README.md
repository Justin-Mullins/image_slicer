# image_slicer

This is a python script that takes an image and splits it into pieces and saves the pieces to the same dirctory.

----------------------
----  HOW TO USE  ----
----------------------

Just run the python image_slicer.py file in the terminal.
It will prompt you to input the full path to where the image is located. For example:  /home/user/Documents/image.jpg
Then it will prompt you for the number of rows and columns you want to split the image into.

The image pieces will be saved to the same directory the original image is located.
